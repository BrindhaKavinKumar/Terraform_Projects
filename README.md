# Terraform Projects

This repository contains multiple real-time infrastructure projects built using **Terraform** on AWS.

Each folder represents an independent Terraform project demonstrating different AWS services and infrastructure setups.

## Projects

* **aws-infrastructure** – Terraform configuration to provision AWS infrastructure such as VPC, subnets, EC2 instances, security groups, and an application load balancer.
* **s3-static-website** – Terraform project for hosting a static website using Amazon S3.

More Terraform projects will be added to this repository as part of learning and building real-world DevOps infrastructure.

## Usage

Navigate to any project folder and run:

terraform init
terraform plan
terraform apply
